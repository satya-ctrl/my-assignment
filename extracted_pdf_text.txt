The "Fact-Check" Agent 
 
The Objective: Build a "Fact-Checking Web App" that automates claim verification.

a PDF, cross-referencing claims against live web data, and flagging inaccuracies. Please find the pdf attached to 
this. 
 
The Task: Build a deployed web app where users upload a PDF for Automated Factchecking. The system must: 
 
Extract: Identify specific claims (stats, dates, financial/technical figures) from the pdf.

Report: Flag claims as Verified (matches data), Inaccurate (e.g., outdated stats), or False (no evidence found).
